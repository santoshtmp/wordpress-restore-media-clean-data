Cleans old/unused WordPress data and restores missing media library files to current location from remote wp site.

== Description ==

Cleans old/unused WordPress data and restores missing media library files to current location from remote wp site.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/restore-media-clean-data/` or install via the Plugins menu.
2. Activate the plugin through the **Plugins** menu in WordPress.
